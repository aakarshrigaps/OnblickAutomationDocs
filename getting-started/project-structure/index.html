    Project Structure

    The project structure is subject to change, refer to the latest source for more up-to-date structure. It is organized to separate test logic from page object definitions. Below is a generalized representation of project structure:

    .
├── README.md
└── HR Modules
    ├── Common_Modules
    ├── Downloads
    ├── Environment_URLs
    │   ├── endpoints-config.json
    │   ├── env-config.json
    │   ├── Environment.cs
    │   └── URLs.cs
    ├── PageObjects
    │   ├── LoginPage.cs
    │   ├── OnboardPage.cs
    │   ├── SettingsPage.cs

    │   └── etc.,
    ├── Reports
    │   ├── Report <dd_MMM_yyy>
    │   │   ├── Screenshots
    │   │   │   ├── Screenshot_<H_mm_ss>.png
    │   │   │   └── etc.,
    │   │   ├── ExtentReport - <dd_MMM_yy - hh_mm>.html
    │   │   └── etc.,
    │   └── etc.,
    ├── TestData
    │   │── Files
